Methodology · Data Pipeline · Open Proof Protocol

How CANOPY Scores

All CANOPY outputs are derived from public-domain satellite data, GBIF occurrence records, and open scientific databases. No proprietary or self-reported data enters the scoring pipeline. Every forest cover change figure is computed from raw satellite imagery — not operator claims.

Step 01
Raw Satellite Acquisition
Sentinel-2 Level-2A atmospherically corrected imagery downloaded from ESA Copernicus Open Access Hub (free, no auth required). Landsat Collection 2 SR accessed via USGS Earth Explorer. GEDI L2A canopy height product from NASA Earthdata. ALOS-2 PALSAR-2 via JAXA G-Portal for SAR-derived canopy density.
ESA Copernicus Open Access Hub · USGS Earth Explorer · NASA Earthdata · JAXA G-Portal
VYRION-METHODOLOGY-v1.0.md →
Step 02
NDVI Change Detection
Normalized Difference Vegetation Index computed per pixel from Sentinel-2 Band 4 (red) and Band 8 (NIR). Change alerts triggered when NDVI drops ≥0.15 within a 5-day comparison window across ≥0.1 ha contiguous area. Cloud masking applied via Sentinel-2 scene classification layer (SCL Band). Alert validated against ALOS-2 SAR to reject false positives from cloud shadow.
Hansen et al. 2013 change detection methodology · Google Earth Engine protocol · NDVI threshold per Zhu et al. 2019
Step 03
Corridor Integrity Index
Per-corridor composite score from four sub-indicators: (1) Canopy continuity (fraction of corridor width with NDVI ≥0.6), (2) Minimum corridor width (GEDI-derived canopy extent), (3) Matrix permeability (proportion of surrounding land cover classified as low-resistance), (4) Active alert density (CANOPY deforestation events within 10km in past 90 days). Scored 0–100, weighted equally.
Beier & Noss 1998 corridor science framework · Landscape Ecology · updated per IUCN Guidelines 2020
Step 04
BII Proxy Computation
Biodiversity Intactness Index proxy for restoration sites computed as: mean GBIF species occurrence count within 5km of site boundary in current year vs PREDICTS database mean abundance for same land cover class in pre-1970 baseline. Normalised to 0–100. Acoustic biodiversity index (ABI) from sound sensor data included for 34 instrumented sites. ABI weighted at 40% for those sites.
PREDICTS database · Natural History Museum London · GBIF API v1 · Alcocer et al. 2022 ABI methodology
VYRION Index · Forest System · 13% Weight

How CANOPY Moves the Index

The Forest system carries 13% of the VYRION Index total — the fourth-highest weighting after Atmosphere, Ocean, and Cryosphere. CANOPY activates four of the six Forest system indicators as live feeds, replacing static annual estimates with satellite-computed values that update on every overpass.

Forest System · F1
Global Forest Cover
Live · Sentinel-2 · 5-day
Canopy cover fraction computed from NDVI threshold classification. Baseline: pre-industrial estimated 5.0B ha at 100% score. Current ~3.04B ha → score contribution depressed 39% from baseline.
Forest System · F2
Deforestation Rate
Live · Sentinel-2 / ALOS-2
Annual net deforestation rate vs pre-industrial background rate. Current −4.7M ha/yr is approximately 30× the pre-agricultural background loss rate. Scores near 0 at current trajectory. Primary driver of Forest system WARNING band.
Forest System · F4
Above-Ground Biomass
Live · GEDI LiDAR · BIOMASS SAR
Canopy height-derived AGB estimate from GEDI L2A product, validated against ESA BIOMASS P-band SAR. Pre-industrial AGB estimated at ~860 GtC. Current ~555 GtC → 35% reduction from baseline. BIOMASS satellite improving estimates in data-scarce tropical regions from 2025.
Forest System · F5
Fragmentation Index
Live · GEDI + Sentinel-2
Landscape fragmentation measured as mean patch size and perimeter-area ratio across all forest patches >1 ha globally. Rising fragmentation scores reduce corridor integrity and amplify biodiversity loss for equivalent cover change. Patch isolation penalised non-linearly in scoring formula.
