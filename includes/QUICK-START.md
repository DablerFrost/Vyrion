# Quick Start Guide for Adding the Floating Orb

This guide provides a quick reference for adding the floating orb to your web pages. Follow the steps below to integrate the floating orb easily.

## Step 1: Include the Floating Orb CSS

Add the following CSS to your `<head>` section:
```html
<link rel="stylesheet" href="https://example.com/path/to/orb.css">
```

## Step 2: Add the Floating Orb Markup

Insert the following HTML where you want the orb to appear:
```html
<div class="floating-orb"></div>
```

## Step 3: Add JavaScript for Orb Functionality

Include the following JavaScript at the bottom of your `<body>` section:
```html
<script src="https://example.com/path/to/orb.js"></script>
```

## Final Touches

Ensure that your styles and scripts are correctly linked, and then refresh your page to see the floating orb in action!
