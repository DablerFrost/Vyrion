Section 04 · Output Layer Only
API Console
Six output endpoints returning VYRION-computed indices only. No raw source data is accessible via API. Institutional access required for protected endpoints.
GET/soil-state
LIVE · STREAMING
Returns SSI composite, moisture trend, and carbon trajectory by bounding box. Protected — requires institutional API key.
GET/regeneration-map
LIVE · STREAMING
Returns RVM time-series output by region. ASSI + RPVS + ERBI composite. Geospatial bounding box supported.
GET/collapse-risk
LIVE · CRITICAL SIGNAL
Returns CRSZ alert layer from all 6 indicators. Flags zones exceeding collapse risk threshold. Priority routing for reinsurance and agricultural risk applications.
GET/agriculture-behavior
LIVE · DATA-ONLY
Returns ASSI output layer — agricultural system structure, resilience classification, and monoculture density mapping by region.
GET/verification-layer
LIVE · CONSCIENCE-LINKED
Returns RPVS output. Cross-referenced against CONSCIENCE corporate pledge registry to verify satellite-observable regenerative practice vs. declared corporate commitments.
GET/corporate-soil-exposure
INSTITUTIONAL · CORPORATE API
Returns supply chain soil risk scoring by corporate entity. Links CRSZ collapse zones to corporate land-use footprints. Portfolio exposure quantification for institutional clients.

Section 07 · System Definition
Core Design Principle

ROOT is not a dataset system. ROOT is a licensed Earth-data transformation engine that converts soil, water, and agricultural signals into verified planetary intelligence products. The transformation is the product. The source data is an input under license. The derived intelligence is wholly owned by VYRION PBC.

ROOT maintains strict separation between source data licensing — governed by original provider terms — and VYRION's proprietary transformation layer, which produces the SSI, RVM, CRSZ, and RPVS outputs. No source data is stored, redistributed, or made accessible through ROOT's output layer.

Canonical Statement · Final · Locked

ROOT is a six-indicator terrestrial intelligence module within the VYRION OS that transforms licensed Earth observation and agricultural datasets into verified, time-series soil stability, regeneration velocity, and collapse risk intelligence products — while maintaining strict separation between source data licensing and proprietary derived planetary intelligence.
