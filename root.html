Section 07 · System Definition
Core Design Principle

ROOT is not a dataset system. ROOT is a licensed Earth-data transformation engine that converts soil, water, and agricultural signals into verified planetary intelligence products. The transformation is the product. The source data is an input under license. The derived intelligence is wholly owned.

ROOT maintains strict separation between source data licensing — governed by original provider terms — and VYRION's proprietary transformation layer, which produces the SSI, RVM, CRSZ, and RPVS outputs. No source data is stored, redistributed, or made accessible through ROOT's output layer.

Canonical Statement · Final

ROOT is a six-indicator terrestrial intelligence module within the VYRION OS that transforms licensed Earth observation and agricultural datasets into verified, time-series soil stability, regeneration velocity, and collapse risk intelligence products — while maintaining strict separation between source data licensing and proprietary derived planetary intelligence.
